Bienvenido al Sistema de Gestión de Tareas

Esta es la página de bienvenida del sistema de gestión de tareas.

Para interactuar con la API, utiliza los siguientes endpoints:

Endpoints disponibles:

  • POST /registro - Registrar un nuevo usuario
  • POST /login - Iniciar sesión
  • GET /tareas - Ver esta página de bienvenida

Para más información, consulta la documentación del proyecto.
